We are very close to see it running.

Now, it's time to start the application.

<CodeGroup>

```bash npm
cd ./boilerplate
npm run dev
```

```bash yarn
cd ./boilerplate
npm run dev
```

```bash pnpm
cd ./boilerplate
npm run dev
```

</CodeGroup>

**You're good to go!** You can deploy it to any cloud service or even use ngrok to share you're application with the rest of the world.
