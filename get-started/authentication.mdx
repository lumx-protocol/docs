---
title: "Authentication"
---

To start using Lumx Protocol, simply make a request to our [create a project](/api-reference/project/auth) endpoint, passing your project's name. You will receive an API key to start making requests to our sandbox environment right away.

When you're ready to start building in a production environment, please contact us at hello@lumxstudios.com or send a message on [our Slack community](https://join.slack.com/t/lumxprotocol/shared_invite/zt-1z920ovmb-V6~OlHpvQSQupe7kJLu5YA) for further instructions.
