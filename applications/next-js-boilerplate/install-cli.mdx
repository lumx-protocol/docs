The create-dapp CLI is tool create by Lumx to help developers to generate a full end-to-end creation in Next.js.

With the CLI you will be able to launch your project in less than 5 minutes!

Its super easy. Let's run it on your machine.

<Steps>
<Step title="Run the CLI">
  <CodeGroup>

    ```bash npm
    npm i @lumx-protocol/create-dapp@latest
    ```

    ```bash yarn
    yarn add @lumx-protocol/create-dapp@latest
    ```

    ```bash pnpm
    pnpm i @lumx-protocol/create-dapp@latest
    ```

  </CodeGroup>
</Step>
<Step title="SEO information">

You will be prompted to enter the necessary information to create your project.

> What's the title of your project?

Enter the desired name for the project. It will be used on the **page title** and on the **folder name**.

> What's the description of your project?

Enter a brief description of the project. It will be used on the **meta description tag**.

</Step>

<Step title="Project information">

Continuing, now you will be prompted to enter the information you made on the **start building** section

> What's your **apiKey**?

You generated on this [step](/guides/start-building/get-an-api-key).

> What's the **contractId**?

This is the contractId you (hopefully!) deployed on the [Your first contract](/guides/start-building/your-first-contract) section.

> What's the **itemTypeId**?

Our first itemTypeId generated together was on the [Mint a token](/guides/start-building/mint-a-token)

Last.. but not least

> Do you want to start your Web3 journey?

Choose this option very carefully.. Do you have a suggestion for new options? [Let us know](https://join.slack.com/t/lumxprotocol/shared_invite/zt-1z920ovmb-V6~OlHpvQSQupe7kJLu5YA)!

</Step>
</Steps>
